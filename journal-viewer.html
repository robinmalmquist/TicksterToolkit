How To Find TJN manually

To obtain a TJN file manually, first look up the EventID and VenueID.

Then go to \\prodstage\gxfs\venue and browse to the specific VenueID.

Enter that folder and then to EventInventory and then enter the folder with the EOG ID.

Here you browse to the specific EventID. In that folder you'll find the TJN file.

Full path to TJN file is \\prodstage\gxfs\venue\[VenueID]\EventInventory\[EOGID]\[EventID]
